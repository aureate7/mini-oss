# Mini OSS —— 基于 gRPC 的对象存储附件服务
本项目目标：设计并实现一个 **基于 gRPC 流式通信的 Mini OSS 服务**，支持：
- 高并发文件上传 / 下载
- 大文件分片（chunk）传输
- 断点续传、幂等重试
- 稳定性与可观测性

## 总体设计目标
### 功能目标
- 提供 **对象上传、下载、查询** 等基础能力
- 支持 **客户端流式上传 / 服务端流式下载**
- 支持 **Range 下载与断点续传**
- 支持 **数据完整性校验**
- 提供基础 **鉴权、日志与指标**

### 非功能目标
- 高并发：支持 ≥100 并发上传/下载
- 高吞吐：在本地/高速链路下尽可能榨干带宽
- 稳定性：支持异常中断后的恢复
- 可观测：具备基础指标输出能力
